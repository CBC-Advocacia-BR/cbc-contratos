Como funciona por trás (as 3 opções): a API do Asaas (POST /v3/paymentDunnings, tipo CREDIT_BUREAU) negativa a cobrança vencida. Nenhum documento é preciso — o boleto vencido já comprova a dívida. O Serasa notifica o cliente por carta com 10 dias para pagar; se pagar, a baixa é automática. Toda negativação fica auditada (quem/quando) e vira nota no lead do Kommo.
✓ Conta PJ (Ltda)✓ Saldo R$ 138 mil p/ tarifas✓ Prova de vida feita✓ Endpoint já ativo (32 negativações)
Requisito de dado por cliente: o Serasa exige telefone + endereço + CEP. Já temos isso em 28 dos 30 (os 2 sem dado ficam bloqueados com aviso "completar cadastro"). Recomendação: começar pela Opção C (menor risco/esforço) e evoluir para o console da B se o volume justificar.
